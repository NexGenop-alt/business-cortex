# Business Cortex Setup Guides

Use this folder for operator runbooks that explain how to install, configure, and run a client-safe Business Cortex deployment.

## Guides

- [`hermes-install-and-runbook.md`](hermes-install-and-runbook.md) — install Hermes on Linux, macOS, Windows, and WSL2; configure provider/tools; create profiles; apply Business Cortex memory boundaries; verify the system.

## Client-safety rule

These setup guides describe reusable deployment patterns only. Do not commit client secrets, real tenant memories, bot tokens, private domains, phone numbers, platform IDs, OAuth files, or internal demo-company data.
